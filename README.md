# Проект: отказоустойчивость и масштабирование MongoDB (шардирование + репликация + кеш)

Репозиторий содержит решения заданий 1–10.  
Для проверки заданий **1–6** ревьюер использует директорию **`sharding-repl-cache`** (в ней реализованы задания 2–4), а также смотрит итоговую схему (задания 1, 5, 6).

---

## Структура репозитория

- `docs` - схемы и документы (задание 1, 5-10)
- `mongo-sharding/` — задание 2 (шардирование, 2 шарда)
- `mongo-sharding-repl/` — задание 3 (шардирование + репликация: 2 шарда × 3 реплики)
- `sharding-repl-cache/` — задание 4 (шардирование + репликация + Redis-кеш)

## Инструкции

В каждой директории представлен README.md-файл с подробным описанием и инструкациями
